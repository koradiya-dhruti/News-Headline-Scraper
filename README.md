# News-Headline-Scraper
A Python-based web scraping application that fetches the latest news headlines from a public news website using HTTP requests and BeautifulSoup. The scraped headlines are saved into a text file for automated data collection.

# Objective
Scrape top headlines from BBC News using Python.

# Features
- Uses requests to fetch webpage
- Parses HTML using BeautifulSoup
- Extracts headline tags
- Saves headlines to a text file

# Tools Used
- Python
- requests
- BeautifulSoup

# How to Run
1. Install dependencies:
   pip install requests beautifulsoup4
2. Run:
   python news_scraper.py
